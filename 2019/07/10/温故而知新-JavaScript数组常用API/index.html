温故而知新--JavaScript数组常用API

  • js

ES6中数组的扩展

1、Array.from()

Array.from用于将两类对象转为真正的数组：类似数组的对象和可遍历的对象（包括ES6中新增的数据结构Set和Map）。

Array.from可以将部署了Iterator接口的数据结构转为数组，同时Array.from方法还支持类似数组的对象，所谓类似数组的对象其本质特性为必须用length属性，任何有length属性的对象，都可以通过Array.from方法将其转为数组，而扩展运算符只能将具有Iterator接口的数据结构转为数组。

2、find()和findIndex()

find()方法用于找出第一个符合条件的数组成员，它的参数是一个回调函数，接受三个参数，依次为当前的值、当前的位置和原数组，所以成员依次执行该回调，找到符合条件的数组成员，则返回该成员，如果没有符合条件的成员，则返回undefined.

findIndex()方法和find()方法非常类似，返回第一个符合条件的数组成员的位置，如果所有都不符合条件，则返回-1。

3、flat()和flatMap()

数组的成员有时还是数组，为了处理数据方便需要把多维数组变为一维数组，Array.prototype.flat()用于将嵌套的数组“拉平”，变成一维的数组。该方法返回一个新数组，对原数组没有影响。

flatMap()方法对原数组的每个成员执行一个函数，然后对返回值组成的数组执行flat()方法。该方法返回一个新数组，不改变原数组。

4、includes()

includes()方法返回一个布尔值，表示某个数组是否包含给定的值，与字符串的includes()方法类似。

改变数组本身

1、pop/push、shift/unshift

pop(）方法是删除数组中的最后一个元素，并且返回这个元素。

push()方法是在数组末尾添加一个或多个元素，并且返回数组新的长度。与pop()是一组相对的方法。

shift()方法是删除数组的第一个元素，并且返回这个元素。与pop()用法相同，只是位置不用

unshift()方法是用于在数组开始的位置添加一个或多个元素，并返回数组新的长度。与push()用法相同，位置不同。

2、reverse、sort

reverse()方法颠倒数组中元素的位置，第一个会变成最后一个，最后一个会变成第一个，该方法返回对数组的引用。

sort()方法是对数组元素进行排序，并返回这个数组。sort()方法中可以跟一个回调函数，按照回调函数的返回值进行排序。sort()方法的默认排序规则是按照数组中各元素转换为字符串的Unicode码排序升序排列。如果指明了排序规则，则按照指定的排序规则排序。

3、splice

splice()方法是用新元素替换旧元素的方式来修改数组。返回值由原数组中删除元素组成的数组。

用法：arr.splice(start,deleteCount, [item1,item2,...])

start 指定从哪一位开始。如果超过数组长度，会从数组末尾开始添加内容；如果为负数，表示从数组末尾开始的第-start位开始添加内容

deleteCount 指定要删除的元素个数，如果为0，表示不删除

[item1,item2,...] 指要新增的元素，如果没有，表示只删除

不改变数组本身

1、concat

concat()方法将传入的数组或者元素与原数组合并，组成一个新的数组并返回。

2、join

join()方法将数组中的所有元素连接成一个字符串，默认用逗号分隔

3、slice

slice()方法将数组中一部分元素浅复制入新的数组对象，并返回这个数组对象。

arr.slice[start,end] start:指定复制开始位置的索引。end有值表示复制结束位置的索引（不包括此位置），如果end不存在，表示从start位置复制到最后

常见的数组遍历方法

1、forEach

forEach()方法指定数组的每项元素都执行一次传入的函数，返回值为undefined

2、map

map()方法遍历数组，使用传入函数处理每个元素，并返回函数的返回值组成的新数组。

3、every/some

every()方法使用传入的函数测试所有元素，只要其中有一个函数返回值为false，那么该方法的结果为false；如果全部返回true，那么该方法的结果才为true。

some()方法刚好同every()方法相反，some()测试数组元素时，只要有一个函数返回值为true，则该方法返回true，若全部返回false，则该方法返回false。

4、filter

filter()方法使用传入的函数测试所有元素，并返回所有通过测试的元素组成的新数组。它就是一个过滤器，筛掉不符合条件的元素。

5、reduce

reduce()方法接收一个方法作为累加器，数组中的每个值（从左至右）开始合并，最终为一个值。
